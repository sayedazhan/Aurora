# Aurora

**The easiest way to send, split and settle money together.**

Aurora is a collaborative financial platform for shared money. It helps friends, families, housemates, clubs and small businesses manage shared expenses, settlements and payments in one simple workspace.

---

## Product Vision

Aurora makes shared money effortless.

---

## Current Status

Aurora is currently in active MVP development.

Completed:

- Marketing website
- Authentication
- Supabase integration
- User profiles
- Workspace dashboard
- Financial Spaces
- Create Space flow
- Space dashboard
- Reusable dashboard components

In progress:
