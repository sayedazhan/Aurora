# Executive Summary

Aurora is a collaborative financial platform that helps people send, split and settle money together.

Aurora combines modern payments, shared financial workspaces, AI assistance and real-time collaboration into a single experience.

Aurora is designed for:

- Friends
- Families
- Couples
- Housemates
- Clubs
- Small Businesses

Aurora is **not** a bank.

Aurora orchestrates payments, collaboration and intelligent financial workflows.
